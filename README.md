# Premo Web - تصميم مواقع إلكترونية احترافية

موقع احترافي لشركة Premo Web المتخصصة في تصميم وتطوير المواقع الإلكترونية.

## 🎯 الخصائص الرئيسية

✅ **تصميم متجاوب (Responsive)** - يعمل بسلاسة على جميع الأجهزة  
✅ **دعم اللغة العربية (RTL)** - تصميم كامل للغة العربية  
✅ **أنيميشن متحرك** - خلفيات ملونة متحركة للمشاريع  
✅ **سهل الصيانة** - كود نظيف وسهل التعديل  
✅ **تحسين محركات البحث** - SEO جاهز  
✅ **سرعة عالية** - تحميل سريع جداً  

## 📚 الأقسام المتضمنة

- **Hero Section** - قسم البداية جذاب مع أنيميشن
- **Services** - عرض الخدمات المختلفة
- **Portfolio** - عرض الأعمال السابقة (مواقع المطاعم والمعمار والصيدليات)
- **Team** - فريق العمل
- **Contact** - معلومات التواصل والواتساب

## 🚀 التقنيات المستخدمة

- **HTML5** - التنسيق الحديث
- **Tailwind CSS** - تصميم متقدم
- **JavaScript** - الوظائف التفاعلية
- **Google Fonts** - خط Cairo العربي
